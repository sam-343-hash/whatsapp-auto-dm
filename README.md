WhatsApp Auto DM Script 🚀

Automates sending WhatsApp messages to multiple contacts.

Features
- Send bulk WhatsApp messages instantly
- Schedule messages at a specific time
- Easy to customize contacts and messages

Requirements
- Python 3.x
- pywhatkit library

How to Run
1. Install dependency: pip install pywhatkit
2. Edit contacts list in the script
3. Open WhatsApp Web in Chrome
4. Run: python whatsapp_auto_dm.py
